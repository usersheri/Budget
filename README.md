The Expense & Budget Manager is a simple yet effective Streamlit web application that helps users record, view, and analyze their expenses.
It demonstrates Object-Oriented Programming (OOP) principles and integrates an SQLite database for storing expense data permanently.

This project provides users with an interactive interface to manage their daily finances and visualize category-wise spending trends.
